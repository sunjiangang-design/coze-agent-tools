# Coze Agent Tools

## Projects

### mfg-vision-qc - Industrial Visual Quality Control Demo

Quick-start template for manufacturing visual QC based on YOLOv8 + Anomalib. Demo scenario: vibrating screen industry.

- YOLOv8 training/inference/export scripts
- Anomalib anomaly detection (normal samples only)
- Vibrating screen industry parameter guide
